# Mobile Gaming Performance Analyzer

A comprehensive tool for analyzing and optimizing mobile gaming performance on REALME devices.

## Features

- **FPS Monitoring**: Real-time frame rate tracking
- **Memory Usage Analysis**: Track RAM and storage usage
- **Battery Impact**: Analyze power consumption during gameplay
- **Temperature Monitoring**: CPU/GPU thermal analysis
- **Performance Reports**: Detailed gaming performance metrics

## Supported Games

- PUBG Mobile
- Call of Duty: Mobile
- Genshin Impact
- Fortnite
- Among Us
- Free Fire

## Installation

```bash
git clone https://github.com/veeravenna/REALME.git
cd REALME
```

## Usage

```bash
python main.py --game "PUBG Mobile"
```

## System Requirements

- REALME Device (Android 10+)
- Python 3.8+
- 2GB RAM minimum
